## Project Overview

Customer churn analysis performed using SQL to identify customer retention patterns and revenue impact.

## Business Problem

Customer churn directly affects revenue and profitability.

The objective of this project is to identify:

* Customer churn rate
* Revenue loss
* High-risk customer segments
* Regional churn trends
